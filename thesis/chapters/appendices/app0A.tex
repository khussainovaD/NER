\section{BIO Annotation Guidelines}

The following rules were applied during manual annotation of the CTS complaint corpus in Label Studio:

\begin{enumerate}
  \item \textbf{\texttt{ROUTE\_NUM}}: Tag any token or token span representing a transit route number. Include the route number digit(s) but not the preceding keyword unless it forms a single lexical unit (e.g., tag \textit{47} in \foreignlanguage{russian}{«Маршрут \underline{47}»}).
  \item \textbf{\texttt{BUS\_ID}}: Tag the vehicle fleet/garage number. This is distinct from \texttt{ROUTE\_NUM} — it identifies the physical vehicle, not the route.
  \item \textbf{\texttt{PLATE\_NUM}}: Tag the full licence plate string including letters and digits.
  \item \textbf{\texttt{STOP\_NAME}}: Tag the full name of the transit stop. Include directional qualifiers when they are part of the official stop name.
  \item \textbf{\texttt{STREET\_NAME}}: Tag the full street/avenue/boulevard name including the street-type word (e.g., \foreignlanguage{russian}{«улица»}, \foreignlanguage{russian}{«шоссе»}).
  \item \textbf{\texttt{STREET\_INTERSECTION}}: Tag the full span describing an intersection, including both street names and the conjunction word (e.g., \foreignlanguage{russian}{«угол Республики и Сейфуллина»}).
\end{enumerate}
